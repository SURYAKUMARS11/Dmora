# Instagram Auto-Reply & Auto-DM Webhook App (InstaResponder)

InstaResponder is a Node.js dashboard application built to automate public comment replies and private direct messages (DMs) on your Instagram Business or Creator page when specific keywords are commented (e.g. `"sent"`, `"pdf"`, `"link"`).

It includes a premium, responsive dark-mode management console with built-in configuration panels, step-by-step setup walkthroughs, dynamic telemetry logs, and a local webhook event simulator to test triggers before deploying or linking live Meta webhooks.

## 🚀 Quick Start (Local Setup)

### 1. Install Dependencies
Make sure you have [Node.js](https://nodejs.org/) installed (v18+ recommended).
Open your terminal in the project directory (`D:\AutoDm`) and run:
```bash
npm install
```

### 2. Run the Application
Start the server in development watch mode:
```bash
npm run dev
```
The server will boot up and be accessible locally:
* **Control Dashboard:** [http://localhost:3000](http://localhost:3000)
* **Webhook Endpoint:** `http://localhost:3000/webhook`

## 🛠️ Step-by-Step Meta Setup Guide

To automate live Instagram comments and DMs, you need to link your Instagram account with a Facebook Page and register a Meta Developer Application. An interactive guide is included in the dashboard's **Setup Guide** tab, but here is a quick summary:

### Step 1: Link Accounts & Enable Message Access
1. Switch your Instagram account to a **Professional Account** (Creator or Business) in Settings.
2. Link it to a **Facebook Page** under your control.
3. **CRITICAL:** In the Instagram app, navigate to *Settings &rarr; Privacy &rarr; Messages* and toggle **"Allow Access to Messages"** to **ON**. This grants the API permission to read/write your inbox messages.

### Step 2: Create a Meta Developer App
1. Go to the [Meta Developer Portal](https://developers.facebook.com/) and create a **Business App**.
2. Add the **Instagram Graph API** product to your app.
3. Add the **Messenger** product to your app (this is used to send direct messages).

### Step 3: Configure Token and Webhooks
1. In the App Dashboard sidebar, navigate to *Messenger &rarr; Instagram Settings*.
2. Connect your Facebook Page and click **Generate Token**. Copy this Page Access Token.
3. Go to [http://localhost:3000](http://localhost:3000), click **Bot Settings**, and paste the token into the **Page Access Token** field.
4. Set up an ngrok tunnel to expose your local port `3000` to the internet (Meta requires an HTTPS endpoint):
   ```bash
   ngrok http 3000
   ```
5. Copy the HTTPS forwarding address generated by ngrok (e.g. `https://xxxx.ngrok-free.app`) and append `/webhook` (e.g. `https://xxxx.ngrok-free.app/webhook`).
6. Register this URL as your Webhook callback in the Meta Developer Portal:
   * **Callback URL:** `https://xxxx.ngrok-free.app/webhook`
   * **Verify Token:** `my_secure_verify_token_123` (or whatever you change it to in the settings page)
7. Subscribe to the **comments** webhook field on the Instagram Graph API.

## 🔒 Security & Loop Prevention
To avoid infinite loops where the bot replies to its own messages:
1. Configure your page's Instagram Username in **Bot Settings**.
2. Turn on the **"Ignore replies that are sub-comments"** setting, which prevents the bot from replying to nested conversations.
